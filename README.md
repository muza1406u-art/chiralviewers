# Artemether Chirality Explorer (Python Tkinter)

This project is a **Python-only desktop application** using Tkinter.
It helps students explore selected **Artemether stereocentres** and view
CIP priority ordering for each center.

## Features

- Pure Python GUI (`app.py`) with Tkinter
- List of stereocentres with fixed R/S configurations
- Detail panel for each center:
  - position and configuration
  - short description
  - CIP priority list (1 → 4)

## Run

```bash
python3 app.py
```

No third-party dependencies are required.
